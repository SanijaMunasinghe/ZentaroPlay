# ZentaroPlay

ZentaroPlay is a browser-based gaming website that provides free, simple, and accessible online games. All games run directly in the web browser, allowing users to play instantly without downloads or installations.

---

## Overview

ZentaroPlay is designed to be lightweight, easy to use, and accessible on modern devices. The project focuses on casual browser games built with standard web technologies.

---

## Features

- Free-to-play browser games  
- No downloads or installations required  
- Responsive design for desktop and mobile  
- Simple and clean user interface  
- Easy to extend with additional games  

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla)  

---

## Getting Started

To run the project locally:

1. Clone or download this repository  
2. Open the `index.html` file in any modern web browser  

No additional setup is required.

---

## Deployment

The website is intended to be hosted using **GitHub Pages** for free public access.
